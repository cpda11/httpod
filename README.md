# httpod

![Build](https://github.com/tgunsch/httpod/workflows/Go/badge.svg)

A simple HTTP Request & Response Service written in go, shamelessly stolen from https://httpbin.org.

## Local run

```shell
go run cmd/main.go
```

## Build docker image

```shell
docker build -t thgunsch/httpod
```
